===================================
Transifex version 0.4 release notes
===================================

Transifex 0.4 is planned to be released in October 2008, and will most likely
be the last stable release from the TG branch.

Transifex 0.5 will most likely be the first release based on the Django branch.

Details still need discussion on the mailing list, and this document will be
updated as soon as we decide on them.
